# Quick Start
The quick start guide will get you a working Irving install using data from the [Irving Example Theme](?path=/docs/wordpress-example-theme--page). Check out the full [tutorial](/?path=/docs/tutorial-introduction--page) for a more in-depth introduction to Irving.

_**Note:** An Irving CLI is on the roadmap which will simplify these instructions._

## Requirements/Assumptions
For this quick start guide, we assume you have,
* Node 12 and NPM 6 installed
* Some basic command line knowledge

## Clone the Create Irving App repo
`git clone https://github.com/alleyinteractive/create-irving-app`

The [Create Irving App](https://github.com/alleyinteractive/create-irving-app) repository has the skeleton for an Irving application.

## Change directories into the site folder
`cd create-irving-app`

## Set environment variables using our example
`cp .env.example .env`

The required environment variables are,
* `ROOT_URL` - URL of the express server. Defaults to `http://localhost:3001`.
* `API_ROOT_URL` - URL of the components endpoint provider. We host this URL for demo/quick start purposes, `https://wordpress.irvingjs.com/wp-json/irving/v1`

## Install Irving
`npm install`

## Run the Irving development server
`npm run dev`

Irving will open a new browser window at `http://localhost:3001` with your site.

## Create a production build
`npm run build`
