# Writing Styles in Irving
Irving offers several methods of writing styles for your project depending on your preferences and needs. The following are built into the Irving ecosystem currently:
* [CSS with CSS Modules](/docs/styling-style-with-css--page) - This is the mode of writing styles that Irving offers out of the box. This is plain ol' vanilla CSS stylesheets with added [CSS Modules functionality via `css-loader`](https://webpack.js.org/loaders/css-loader/#modules).
* [Styled Components](/docs/styling-style-with-styled-components--page) - Irving supports styling with [`styled-components`](https://styled-components.com/) by way of the [`@irvingjs/styled`](https://www.npmjs.com/package/@irvingjs/styled) package and extended with a component library via [`@irvingjs/styled-components`](https://www.npmjs.com/package/@irvingjs/styled-components) to help you get your site up and running quickly.
* [PostCSS](/docs/styling-style-with-postcss--page) - Irving supports PostCSS via the [`@irvingjs/postcss`](https://www.npmjs.com/package/@irvingjs/postcss) package, allowing you to leverage the wealth of plugins in the PostCSS ecosystem to further optimize your stylesheets.

Navigate to the links above to find out more about installing and/or using each of the above styling methods.
