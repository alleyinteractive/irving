# Term Link
Get the term link.

This component returns an `irving/link` component with `href` set to the `link` property of the WP Term object. Term ID can be set by context or `term_id`.

Renders as the `Link` styled component [(docs)](?path=/docs/styled-components-link--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/term-link).

## Examples

Setting `term_id` directly.
```json
{
  "name": "irving/term-link",
  "config": {
    "term_id": 1
  }
}
```

Setting `term_id` via provider.
```json
{
  "name": "irving/term-provider",
  "config": {
    "term_id": 1
  },
  "children": [
    {
      "name": "irving/term-link"
    }
  ]
}
```

Both will render as,
```json
{
  "name": "irving/text",
  "config": {
    "href": "https://example.com/category/uncategorized"
  }
}
```
