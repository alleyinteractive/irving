# Image
Output an image.

Base image component supporting various display options including alt, src, srcset, lazy loding, fallback, captions, credits, and more.

Renders as the `Image` styled component [(docs)](?path=/docs/styled-components-image--basic-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/image).

## Example
```json
{
	"name": "irving/image",
	"config": {
		"src": "https://placekitten.com/408/287"
	}
}
```
