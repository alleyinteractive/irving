# Fragment
Render children in a React fragment or any other HTML tag.

Use a React fragment to render children. Optionally render as an html element to include inline styles or a theme.

Renders the `Fragment` styled component [(docs)](?path=/docs/styled-components-fragment--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/fragment).

## Example
```json
{
	"name": "irving/fragment",
	"children": [
		{
			"name": "irving/text",
			"config": {
				"content": "Example Fragment"
			}
		}
	]
}
```
