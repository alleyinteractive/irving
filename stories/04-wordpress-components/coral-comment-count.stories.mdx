# Coral Comment Count
Coral comments count.

Displays the number of comments in a given coral thread.

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/coral-comment-count).

## Config
```json
{
  "article_URL": {
    "default": "",
    "type": "string"
  },
  "embed_URL": {
    "default": "",
    "type": "string"
  },
  "no_text": {
    "default": false,
    "type": "bool"
  },
  "post_id": {
    "default": 0,
    "type": "int",
    "hidden": true
  }
}
```

## Example
```json
{
	"name": "irving/coral-comment-count"
}
```
