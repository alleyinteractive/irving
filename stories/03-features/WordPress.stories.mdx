# WordPress Features
This section documents WordPress features that are supported in Irving.

## Custom Logo
Upload your logo via the Customizer, and use the `irving/site-logo` [(docs)](?path=/docs/wordpress-components-site-logo--page) component in your template.

[WordPress Documentation](https://developer.wordpress.org/themes/functionality/custom-logo/)

## Favicon
Upload your favicon to the Customizer, and Irving handles the rest.

[WordPress Documentation](https://wordpress.org/support/article/creating-a-favicon/)

## Feeds
WordPress feeds are supported via [proxy](?path=/docs/features-proxies--page).

## Previews
To enable previews, you must install and enabled the [JWT Auth plugin](http://localhost:6006/?path=/docs/integrations-supported-wordpress-plugins--page#jwt-auth).

Coming soon, unauthenticated previews.

## Sitemaps
Sitemaps via Jetpack, MSM, or WordPress Core are supported via [proxy](?path=/docs/features-proxies--page).
