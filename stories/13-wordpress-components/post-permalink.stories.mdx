# Post Permalink
Get the post permalink.

Renders as the `Link` styled component [(docs)](?path=/docs/styled-components-link--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/post-permalink).

## Overview
Used to display the post permalink in any post context. It uses the `irving/link` component by setting the `href` config value.

### Notes
* This component wraps `get_the_permalink()` [(docs)](https://developer.wordpress.org/reference/functions/get_the_permalink/).

## Config
```json
{
  "href": {
    "default": "",
    "type": "string"
  },
  "fragment": {
    "default": "",
    "type": "string"
  },
  "rel": {
    "default": "",
    "type": "string"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "target": {
    "default": "",
    "type": "string"
  }
}
```

## Examples

### Basic example
Template:
```json
{
  "name": "irving/post-permalink",
  "children": [
    {
      "name": "irving/post-title"
    }
  ]
}
```

Endpoint output:
```json
{
  "name": "irving/post-permalink",
  "_alias": "irving/link",
  "config": {
    "href": "https://example.com/about-us/"
  }
}
```
