# Button
Render Button component.

Renders the `Button` styled component [(docs)](?path=/docs/styled-components-button--basic-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/main/inc/components/components/button).

## Overview
Render a Button.

## Config
```json
{
  "tag": {
    "default": "button",
    "type": "string"
  }
}
```

## Examples
See the [button styled component documentation](?path=/docs/styled-components-button--basic-example) for more examples.

### Basic Example
```json
{
  "name": "irving/button",
  "children": [
    {
      "name": "irving/text",
      "config": {
        "content": "Cool Button"
      }
    }
  ]
}
```
