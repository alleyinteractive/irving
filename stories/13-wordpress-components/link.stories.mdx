# Link
Custom anchor element implementation

Renders as the `Link` styled component [(docs)](?path=/docs/styled-components-link--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/link).

## Overview
Base anchor/link component which ensures all links use the [History API](https://developer.mozilla.org/en-US/docs/Web/API/History_API) to navigate relative paths, bypassing a server side render.

## Config
```json
{
  "href": {
    "default": "",
    "type": "string"
  },
  "fragment": {
    "default": "",
    "type": "string"
  },
  "rel": {
    "default": "",
    "type": "string"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "target": {
    "default": "",
    "type": "string"
  }
}
```

## Examples

### Basic example
```json
{
  "name": "irving/link",
  "config": {
    "href": "/about-us/",
    "rel": "nofollow noreferrer",
    "target": "_blank"
  }
}
```
