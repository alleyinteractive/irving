# Styled CSS Grid Cell
Grid cell provided by the `styled-css-grid` package.

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/styled-css-grid-cell).

## Overview
See https://styled-css-grid.js.org/.

## Config
```json
{
  "area": {
    "description": "The grid-area CSS property. Not provided by default.",
    "type": "string"
  },
  "center": {
    "description": "Horizontally align the text contents of the cell. Default is false.",
    "type": "string"
  },
  "height": {
    "description": "Cell height in units, default is 1.",
    "type": "number"
  },
  "left": {
    "description": "The grid-column-start CSS property. Not provided by default.",
    "type": "string"
  },
  "middle": {
    "description": "Vertically align the contents of the cell. Default is false.",
    "type": "string"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "top": {
    "description": "The grid-row-start CSS property. Not provided by default.",
    "type": "string"
  },
  "width": {
    "description": "Cell width in units, default is 1.",
    "type": "number"
  }
}
```

## Examples

### Basic example
```json
{
  "name": "styled-css-grid/cell"
}
```
