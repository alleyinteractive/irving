# Site Menu
Output links as menu items.

Renders as the `Menu` styled component [(docs)](?path=/docs/styled-components-menu--basic-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/main/inc/components/components/site-menu).

## Overview
This component uses the `location` config value with `get_nav_menu_locations()` [(docs)](https://developer.wordpress.org/reference/functions/get_nav_menu_locations/) and `wp_get_nav_menu_object` [(docs)](https://developer.wordpress.org/reference/functions/wp_get_nav_menu_object/) to recursively build the menu with children `menu-item`.

## Config
```json
{
  "display_name": {
    "type": "bool",
    "default": false
  },
  "location": {
    "default": "",
    "type": "string"
  },
  "menu_id": {
    "type": "int"
  },
  "menu_name": {
    "type": "string",
    "default": "Default"
  }
}
```

## Examples

### Basic example
```json
{
  "name": "irving/site-menu",
  "config": {
    "location": "header-primary"
  }
}
```
