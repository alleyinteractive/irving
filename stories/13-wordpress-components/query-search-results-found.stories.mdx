# Query Search Results Found
Display meta information about the search results.

Renders as the `Text` styled component [(docs)](?path=/docs/styled-components-text--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/query-search-results-found).

## Overview
This component allows you to display some meta information about a WP_Query used for search.

### Notes
The config value `content_format` is a wrapper around `sprintf()` [(docs)](https://www.php.net/manual/en/function.sprintf.php) with two available parameters,
* `%1$s` - Number of results.
* `%2$s` - Search string.
* `%3$s` - Current page.
* `%4$s` - Number of pages found.

## Config
```json
{
  "content_format": {
    "default": "%1$s results found for \"%2$s\"",
    "type": "string"
  },
  "content": {
    "default": "",
    "type": "string"
  },
  "wp_query": {
    "hidden": true,
    "type": "object"
  }
}
```

## Examples

### Basic example

Template:
```json
{
  "name": "irving/query-search-results-found"
}
```

Endpoint output:
```json
{
  "name": "irving/query-search-results-found",
  "_alias": "irving/text",
  "config": {
    "content": "18 results found for \"Test\""
  }
}
```

### Customized Message

Template:
```json
{
  "name": "irving/query-search-results-found",
  "config": {
    "content_format": "%1$s results found for \"%2$s\". Currently on page %3$s of %4$s."
  }
}
```

Endpoint output:
```json
{
  "name": "irving/query-search-results-found",
  "_alias": "irving/text",
  "config": {
    "content": "18 results found for \"Test\". Currently on page 1 of 2.",
    "content_format": "%1$s results found for \"%2$s\". Currently on page %3$s of %4$s."
  }
}
```
