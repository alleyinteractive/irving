# Image
Output an image.

Renders as the `Image` styled component [(docs)](?path=/docs/styled-components-image--basic-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/image).

## Overview
A comprehensive abstraction for images. In addition to properties supported by `<img/>`, we add support for meta (caption & credit) and fallback images.

## Examples
See the [image styled component documentation](?path=/docs/styled-components-image--basic-example) for more examples.

### Basic example
```json
{
  "name": "irving/image",
  "config": {
    "alt": "A kitten",
    "src": "https://placekitten.com/408/287"
  }
}
```

### Getting an image by WordPress attachment ID
__Note:__ Use the context provider to set this property more effectively.
```json
{
  "name": "irving/image",
  "config": {
    "id": 1
  }
}
```
