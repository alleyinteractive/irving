# Site Logo
Display the site name or logo.

Renders as the `Logo` styled component [(docs)](?path=/docs/styled-components-logo--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/site-logo).

## Overview
Uses the `custom_logo` theme mod [(docs)]() to output either the site's name, or the logo.

### Notes
* This component needs some refactoring and updating.

## Config
```json
{
  "href": {
    "default": "/",
    "type": "string"
  },
  "logo_image_url": {
    "default": "",
    "type": "string"
  },
  "site_name": {
    "default": "",
    "type": "string"
  }
}
```

## Examples

### Basic example
```json
{
  "name": "irving/site-logo"
}
```
