# Container
Base wrapper component for creating layouts.

Renders the `Container` styled component [(docs)](?path=/docs/styled-components-container--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/container).

## Overview
This component is used to wrap other components, providing structure and styles as needed. It allows you to easily center components and define a max width.

### Notes
* `background_image_url` will be rendered as the `background-image: url()` property.
  * It will also set `background-repeat: no-repeat` and `background-size: cover;`.
* `max_width` accepts breakpoint keys, a string value (ex. `"6rem"`), or a pixel value integer (ex. `600` becomes `600px`).
  * Default options are `xs` (0px), `sm` (600px), `md` (960px), `lg` (1280px), and `xl` (1920px).

## Config
```json
{
  "background_image_url": {
    "default": "",
    "type": "string"
  },
  "max_width": {
    "default": "",
    "type": "string"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "tag": {
    "default": "div",
    "type": "string"
  }
}
```

## Examples
See the [container styled component documentation](?path=/docs/styled-components-container--basic-example) for more examples.

### Basic Example
```json
{
  "name": "irving/container",
  "theme": "body_wrapper"
}
```

### Using `background_image_url`
```json
{
  "name": "irving/container",
  "config": {
    "background_image_url": "https://placekitten.com/408/287"
  }
}
```

### Using `max_width`
Using breakpoint config,
```json
{
  "name": "irving/container",
  "config": {
    "max_width": "lg"
  }
}
```

Max width of `6rem`,
```json
{
  "name": "irving/container",
  "config": {
    "max_width": "6rem"
  }
}
```

Max width of `600px`,
```json
{
  "name": "irving/container",
  "config": {
    "max_width": "600"
  }
}
```
