# Post Featured Image
Get the post featured image.

Renders as the `Image` styled component [(docs)](?path=/docs/styled-components-image--basic-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/post-featured-image).

## Overview
Used to display the [post thumbnail](https://codex.wordpress.org/Post_Thumbnails) in any post context. It uses the `irving/image` component to achieve this.

### Notes
* We rely on WordPress' core image logic to determine the alt, caption, src, and srcset values.

## Examples

### Basic Example
Template:
```json
{
  "name": "irving/post-featured-image",
  "config": {
    "object_fit": "cover"
  }
}
```

Endpoint output:
```json
{
  "name": "irving/post-featured-image",
  "_alias": "irving/image",
  "config": {
    "alt": "A kitten",
    "src": "https://placekitten.com/408/287"
  }
}
```
