# Query Pagination
Pagination UI for a query.

Renders as the `Pagination` styled component [(docs)](?path=/docs/styled-components-pagination--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/query-pagination).

## Overview
Displays a pagination component when used in the context of a `irving/wp_query` context provider.

### Notes
* `irving/wp_query` is one of the default context providers, and will be set using the global `$wp_query` on every template. The `irving/post-list` component will override this global default when used.

## Config
```json
{
  "base_url": {
    "type": "string"
  },
  "current_page": {
    "default": 1,
    "type": "int"
  },
  "display_first_and_last": {
    "default": true,
    "type": "bool"
  },
  "display_prev_and_next": {
    "default": true,
    "type": "bool"
  },
  "pagination_format": {
    "default": "page/%1$d/",
    "type": "string"
  },
  "range": {
    "default": 5,
    "type": "number"
  },
  "style": {
    "default": [],
    "type": "object"
  },
  "total_pages": {
    "default": 1,
    "type": "int"
  },
  "wp_query": {
    "hidden": true,
    "type": "object"
  }
}
```
