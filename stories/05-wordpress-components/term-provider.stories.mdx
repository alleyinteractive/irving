# Term Provider
Term context provider.

Renders as the `Fragment` styled component [(docs)](?path=/docs/styled-components-fragment--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/term-provider).

## Overview
Acts as a context provider for `term_id` to faciliate the use of other term components.

### Notes
Coming soon.

## Config
```json
{
  "term_id": {
    "default": 0,
    "type": "integer"
  }
}
```

## Examples

### Basic example
```json
{
  "name": "irving/term-provider",
  "config": {
    "term_id": 1
  },
  "children": [
    {
      "name": "irving/term-name"
    }
  ]
}
```
