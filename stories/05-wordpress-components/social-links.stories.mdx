# Social Links
Display a series of social media icons with links.

Renders as the `SocialLinks` styled component (docs needed).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/social-links).

## Overview
This component displays icons for social media platforms.

### Notes
Use the `wp_irving_social_links_platforms` filter to modify the `platforms` config.

The icon for each platform will be automatically mapped from `irving/${platform}-icon`. You must ensure this component exists in your component map for each platform key.

## Config
```json
{
  "platforms": {
    "type": "array",
    "default": []
  }
}
```

## Examples

### Basic example
```json
{
  "name": "irving/social-links",
  "config": {
  	"platforms": {
  	  "facebook": "https://www.facebook.com/your-profile",
  	  "twitter": "https://twitter.com/your-profile"
  	}
  }
}
```
