# Site Info
Displays information about the current site.

Renders as the `Text` styled component [(docs)](?path=/docs/styled-components-text--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/site-info).

## Overview
Wrapper for `get_bloginfo()` [(docs)](https://developer.wordpress.org/reference/functions/get_bloginfo/).

## Config
```json
{
  "content": {
    "default": "",
    "type": "string"
  },
  "oembed": {
    "default": false,
    "type": "bool"
  },
  "show": {
    "default": "name",
    "type": "string"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "tag": {
    "default": "span",
    "type": "string"
  }
}
```

## Examples

### Get site name
Template:
```json
{
  "name": "irving/site-info",
  "config": {
    "show": "name"
  }
}
```

Endpoint output:
```json
{
  "name": "irving/site-info",
  "_alias": "irving/text",
  "config": {
    "content": "My cool WordPress site",
    "show": "name"
  }
}
```
