# Styling `irving/post-title`
In this section, we'll style the `irving/post-title` component.

We do this by creating a new component theme.

In order to apply new themes to an `irving` component we need to create a new component theme.

1. create a new theme, which is an object of styled components.
1. add it to your component map
1. update the template

### Create the theme file
We'll start by creating a new file at `components/text/singleTitle.js`.

This file is going to export one or more Styled Components which correspond how the component was built. In addition to the components that ship in `@irvingjs/styled-components`, there are also themes which provide some basic styles.

We'll be introducing a CLI command to help speed up this process.

For our title, we want to start with the `h1` theme ([link](https://github.com/alleyinteractive/irving/blob/main/packages/styled-components/components/text/themes/h1.js)) provided by @irvingjs.
```js
/* eslint max-len: 0 */
import styled from 'styled-components';

// eslint-disable import/prefer-default-export.
export const TextWrapper = styled.div`
  font-size: 2.25rem;
  margin-bottom: 1rem;
`;
// eslint-enable.
```

As we can see in the component, this Styled Component wraps the text rendered.

[docs](https://github.com/alleyinteractive/irving/blob/main/packages/styled-components/components/text/index.js)

```
return (
  <TextWrapper
    {...standardProps}
  >
    {content}
  </TextWrapper>
);
```


```js
/* eslint max-len: 0 */
import styled from 'styled-components';
import { rem } from '@irvingjs/styled/utils';

export const TextWrapper = styled.div`
  font-size: 4rem;
  letter-spacing: -0.015em;
  line-height: 1.2;
  margin-bottom: \${rem(16)};
`;
```

### Include it in the component map

`componentMap.js`
```js
import createThemedComponentMap from '@irvingjs/styled-components/componentMap';
import { componentMap as WordPressComponentMap } from '@irvingjs/wordpress';
import { IntegrationsManager } from '@irvingjs/integrations';

// Import custom themes.
import * as singleByline from 'components/byline/singleByline';
import * as singleTitle from 'components/text/singleTitle';

const themes = {
  'irving/byline': {
    singleByline,
  },
  'irving/text': {
    singleTitle,
  },
};

export default {
  ...WordPressComponentMap,
  ...createThemedComponentMap(themes),
  'irving/integrations': IntegrationsManager,
};
```

### Update `single.json`
`components/text/singleTitle.js`
```json
{
  "name": "irving/post-title",
  "theme": "single_title",
  "config": {
    "tag": "h2"
  }
}
```
