### Register the component in WP Irving
We could stop here if we wanted to. The newsletter component is rendering, and you can write any JS you want to get it working.

And if we wanted to pass some config values around, we can set those directly in the template JSON.

```json
{
	"name": "tutorial/newsletter",
	"config": {
		"position": "footer"
	}
}
```

WP Irving

`/your-theme/components/newsletter/component.php`
```php
<?php
/**
 * Newsletter.
 *
 * @package Your_Theme
 */

namespace Your_Theme;

use WP_Irving\Components;

/**
 * Register the component and callback.
 */
Components\register_component_from_config(
	__DIR__ . '/component',
	[
		'config_callback' => function ( array $config ): array {
			// Modify some config values.
			return $config;
		},
		'children_callback' => function ( array $children, array $config ): array {
			// Modify children components
			return $children;
		},
	]
);
```
