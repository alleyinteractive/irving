# Post Provider
Post context provider.

Renders as the `Fragment` styled component [(docs)](?path=/docs/styled-components-fragment--basic-eample).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/post-provider).

## Overview
Acts as a context provider for `post_id` to facilitate the use of other post components.

### Notes
* `irving/post_id` is one of the default context providers, and will be set using the global `get_the_ID()` on every template. This replaces the need for a `irving/post-provider` on some templates (ex. `single.json`, `page.json`).

## Config
```json
{
  "post_id": {
    "default": 0,
    "type": "integer"
  }
 }
```

## Examples

### Basic example
```json
{
  "name": "irving/post-provider",
  "config": {
    "post_id": 1
  },
  "children": [
    {
      "name": "irving/post-title"
    }
  ]
}
```
