# Post Excerpt
Get the post excerpt.

Renders as the `Text` styled component [(docs)](?path=/docs/styled-components-text--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/post-excerpt).

## Overview
Used to display the [post excerpt](https://wordpress.org/support/article/excerpt/) in any post context. It uses the `irving/text` component by setting the `content` config value.

### Notes
* This component wraps `get_the_excerpt()` [(docs)](https://developer.wordpress.org/reference/functions/get_the_excerpt/).

## Config
```json
{
  "content": {
    "default": "",
    "type": "string"
  },
  "html": {
    "default": false,
    "type": "bool"
  },
  "oembed": {
    "default": false,
    "type": "bool"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "tag": {
    "default": "span",
    "type": "string"
  }
}
```

## Examples

### Basic example
Template:
```json
{
  "name": "irving/post-excerpt"
}
```

Endpoint output:
```json
{
  "name": "irving/post-excerpt",
  "_alias": "irving/text",
  "config": {
    "content": "Lorem ipsum dolor sit amet.",
    "html": false,
    "oembed": false,
    "tag": "span"
  }
}
```
