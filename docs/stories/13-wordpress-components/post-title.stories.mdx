# Post Title
Get the post title.

Renders as the `Text` styled component [(docs)](?path=/docs/styled-components-text--plain-text-example).

[View Source](https://github.com/alleyinteractive/wp-irving/tree/master/inc/components/components/post-title).

## Overview
Used to display the post title in any post context. It uses the `irving/text` component by setting the `content` config value.

### Notes
* This component wraps `get_the_title()` [(docs)](https://developer.wordpress.org/reference/functions/get_the_title/).

## Config
```json
{
  "content": {
    "default": "",
    "type": "string"
  },
  "html": {
    "default": false,
    "type": "bool"
  },
  "oembed": {
    "default": false,
    "type": "bool"
  },
  "style": {
    "default": [],
    "type": "array"
  },
  "tag": {
    "default": "span",
    "type": "string"
  }
}
```

## Examples

### Basic example
Template:
```json
{
  "name": "irving/post-title"
}
```

Endpoint output:
```json
{
  "name": "irving/post-title",
  "_alias": "irving/text",
  "config": {
    "content": "About Us",
    "html": false,
    "oembed": false,
    "tag": "span"
  }
}
```
