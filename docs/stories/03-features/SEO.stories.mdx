# Search Engine Optimization
Since the markup of your site is 100% dictated by the components you decide to build and how you write your markup, Irving leaves SEO decisions and considerations to you.

## Concerns about SEO on Single Page Applications
There is a lot of information about SEO in the context of a SPA. There's debate around how large of an issue this still is, but those points are moot considering Irving's [Server Side/Isomorphic Rendering](?path=/docs/concepts-server-side-rendering--page).

## Yoast
WP Irving has support for [Yoast](?path=/docs/integrations-supported-wordpress-plugins--page#yoast) and extends/improves WordPress' core SEO capabilities.
