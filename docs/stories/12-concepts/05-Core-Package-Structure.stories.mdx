# Irving Core Structure
Irving and all of its packages are managed using [Lerna](https://github.com/lerna/lerna). All packages have their own directory located in the [packages directory](https://github.com/alleyinteractive/irving/tree/main/packages). This documentation page will detail only the code structure of the Core irving package and the root Irving project directory. The anatomy of an extension package can be found in the docs for [creating your own package](?path=/docs/packages-creating-a-package--page).

### Root Files
These are mostly config files.
* `.editorconfig` - [Editor config](https://editorconfig.org).
* `.gitignore` - Files for Git to ignore.
* `.travis.yml` - [Travis CI](http://travis-ci.com) configuration.
* `.eslintignore` - Files for ESLint to ignore.
* `.eslintrc.js` - Global eslint configuration for all Irving packages. [ESLint config](https://eslint.org/docs/user-guide/configuring).
* `jest.config.base.js` - Base jest configuration that all packages extend. This allows us to DRY out our jest setup while still maintaining a configuration file at the root of each package.
* `lerna.json` - Lerna configuration for the Irving repo.

### Core files
* .eslintignore - Files for ESLint to ignore.
* .eslintrc.js - Eslint config for Irving core and projects using it.
* README.md - Intro doc.
* babel.config.js - [Babel config](https://babeljs.io).
* jest.config.js - [Jest config](https://jestjs.io).
* nodemon.js - [Nodemon setup](https://nodemon.io).
* package-lock.json - [NPM Package Lock](https://docs.npmjs.com/files/package-lock.json).
* package.json - [NPM packages](https://www.npmjs.com).

### Core directories
Each folder has a readme with more specific information.
* [Actions](https://github.com/alleyinteractive/irving/tree/main/packages/core/actions) - Redux actions.
* [Assets](https://github.com/alleyinteractive/irving/tree/main/packages/core/assets) - Static assets such as svgs, fonts, and style mixins.
* [Client](https://github.com/alleyinteractive/irving/tree/main/packages/core/client) - Root directory for client-side webpack bundle. Responsible for app hydration and client-side rendering.
* [Components](https://github.com/alleyinteractive/irving/tree/main/packages/core/components) - Core components.
* [Config](https://github.com/alleyinteractive/irving/tree/main/packages/core/config) - Configuration files.
* [Hooks](https://github.com/alleyinteractive/irving/tree/main/packages/core/hooks) - Various React Hooks that can be helpful to development.
* [Reducers](https://github.com/alleyinteractive/irving/tree/main/packages/core/reducers) - Redux Reducers.
* [Sagas](https://github.com/alleyinteractive/irving/tree/main/packages/core/sagas) - React Sagas.
* [Selectors](https://github.com/alleyinteractive/irving/tree/main/packages/core/selectors) - Reselect selectors.
* [Services](https://github.com/alleyinteractive/irving/tree/main/packages/core/services) - Services.
* [Styleguide](https://github.com/alleyinteractive/irving/tree/main/packages/core/styleguide) - See [Styleguide](?path=/docs/packages-styleguide--page).
* [Utils](https://github.com/alleyinteractive/irving/tree/main/packages/core/utils) - Misc utility functions.
